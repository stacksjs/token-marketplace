# Stacks X-Ray

Visualize & debug your application logs.

## ☘️ Features

- Works in browser & server
- Visualizes logs
- Debugs logs

## 🤖 Usage

```bash
bun install -d @stacksjs/x-ray
```

Now, you can use it in your project:

```ts
import xRay from '@stacksjs/x-ray'

// wip
```

Learn more in the docs.

## 🧪 Testing

```bash
bun test
```
