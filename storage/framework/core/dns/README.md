# Stacks Domains

Easily manage your domains.

## ☘️ Features

- DNS management
- Domain management
- Custom email addresses
- SSL management

## 🤖 Usage

```bash
bun install -d @stacksjs/dns
```

Now, you can use it in your project:

```js
import { domains } from '@stacksjs/dns'
```
