# Stacks Server

## ☘️ Features

- easily access the server's configuration

## 🤖 Usage

```bash
bun install -d @stacksjs/server
```

Now, you can use it in your project:

```js
import { server } from '@stacksjs/server'

console.log('server config is', server)
```

Learn more in the docs.

## 🧪 Testing

```bash
bun test
```
