# Stacks Enums

Stacks enums.

## ☘️ Features

- a collection of Stacks relevant enums

## 🤖 Usage

```bash
bun install -d @stacksjs/enums
```

To view the full documentation, please visit [https://stacksjs.org/enums](https://stacksjs.org/enums).

## 🧪 Testing

```bash
bun test
```
