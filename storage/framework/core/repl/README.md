# REPL

A Stacks optimized REPL.

## ☘️ Features

- **REPL** - A Stacks optimized REPL.

## 🤖 Usage

```bash
bun install -d @stacksjs/repl
```

Now, you can use it in your project:

```js
import { repl } from '@stacksjs/repl'

// wip
```
