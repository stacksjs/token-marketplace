# Stacks Alias

The Stacks aliases.

## ☘️ Features

- 🤷🏼‍♂️ git Diff Interpreter

## 🤖 Usage

```bash
bun install -d @stacksjs/alias
```

You may now use:

```ts
import { alias } from '@stacksjs/alias'

console.log(alias)
```

Learn more in the docs.

## 🧪 Testing

```bash
bun test
```
